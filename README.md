 Marketing Campaign Data Analysis & Dashboard
 Project Overview

This project focuses on analyzing marketing campaign performance using Excel-based data analysis and visualization. The goal is to understand campaign effectiveness through key metrics and present insights using an interactive dashboard.

README.md → Project documentation
 Key Objectives
Analyze marketing campaign performance
Identify trends in engagement and conversions
Evaluate campaign effectiveness across different parameters
Visualize insights using a dashboard

Data Visualization Techniques

 Key Insights
Campaign performance varies significantly across channels and audience segments
Engagement and conversion trends highlight the most effective strategies
Visual dashboard helps in quick decision-making and performance tracking
 Note

This project is created as part of a data analytics learning internship task.
The dataset is used strictly for educational purposes, and all insights are based on exploratory analysis only.
